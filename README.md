# TacoBoutIt

[Live!](http://tacobout-it.herokuapp.com/#/)

TacoBoutIt is a clone project of Yelp. I have created funtionality such as User Auth, visting Business pages, and creating unique Reviews under each Business.

## Technologies Used

- Ruby on Rails
- AWS S3
- PostgresSql
- Redux
- JavaScript
- React
- HTML
- CSS
- FontAwesome
- Google Maps API
